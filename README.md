Titulo: Modelos de reaseguro financiero para riesgo de pérdidas agrícolas por inundaciones mediante aprendizaje automático y profundo
Este posee los scripts realizados para el desarrollo del proyecto de grado, posee tanto la exporación, preparación, modelado de los datos y los resultados de los mismos.


Contenido:
1. Preparación_eventos_UNGRD_2010_2024.ipynb: Este script contiene la extracción y preparación de los eventos de la institución Unidad Nacional De Gestión de Riesgos de Desastres (UNGRD).
2. Preparación _frontera_agrícola.ipynb: Este script contiene la extracción y preparación de los datos obtenidos en UPRA con la herramenta SIPRA de datos agrícolas, como la frontera, restricción, y aptitudes del Maíz y Arroz.
3. Preparación Geoespacial_Clima.ipynb: Este script con tiene la extracción y preparación de los los eventos del clima CHIRPS y además de los datos topográficos obtenidos en Googe Google Earth Engine (GEE).
4. Cartografia_Col.ipynb: Este scrpt es la exploración y preparación de la cartográfia de Colombia, esto ayuda a la filtración de los municipios.
5. Resultados_sin_proceso_2025_3.ipynb: Este script contiene la exploración y análisis de datos obtenidos como inundaciones, ríos, quejas, y el resultado de la preparación de los datos de la UNGRD.
6. Analisisestadistico_r: Este script de R comprende el desarrollo del análisis estadístico de datos continuos del dataset obtenido.
7. OE_1-RF_U-NET.ipynb: Este script contiene la ejecución del Objetivo Especifico 1 que es el desarrollo y resultado de los modelos de Random-Forest y U-net.
8. OE_2-LSTM.ipynb: Este script contiene la ejecución del Objetivo Especifico 2 que comprende el desarrollo y resultado de modelos LSTM.
9. OE_3-Diseño_prelimienar_del_reaseguro_parametrico.ipynb: Este script comprende el desarrollo del Objetivo Especifico 3.

Por otro lado, también están los resultados de los modelados de los objetivos especificos 1, 2 y 3.
